sequenceDiagram

    participant SYS as School Ordering System

    RM->>SYS: login
    SYS-->>RM: display main page

    RM->>SYS: open Manage Menu
    SYS-->>RM: display MenuList

    alt add menu
        RM->>SYS: add menu
        SYS-->>RM: display updated MenuList
    else edit menu
        RM->>SYS: edit menu
        SYS-->>RM: display updated MenuList
    else delete menu
        RM->>SYS: delete menu
        SYS-->>RM: display updated MenuList
    end
