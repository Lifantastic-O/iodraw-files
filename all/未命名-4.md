```mermaid
sequenceDiagram
    actor Passenger
    participant FE as UniGrab System Front End
    participant BE as System/Backend
    participant 

    Passenger->>FE: Select pickup point and destination
    FE->>BE: Transfer location data
    BE-->>FE: Return price estimation
    FE-->>Passenger: Display fare request

    Passenger->>FE: Confirm order
    FE->>BE: Send ride request
    BE-->>FE: Return driver list
    FE-->>Passenger: Display driver list

    Passenger->>FE: Select driver
    FE->>BE: Connect the driver
    BE->>Driver: Send ride request

    alt Driver accepts
        Driver-->>BE: Accept ride
        BE-->>FE: Return confirmation info
        FE-->>Passenger: Display booking result
    else Driver rejects / no response
        Driver-->>BE: Reject ride / No response
        BE-->>FE: Return driver reject
        FE-->>Passenger: Display driver reject result
        FE-->>Passenger: Return to select driver page
    end
```
